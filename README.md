# todo-mithril
A simple todo app with Mithril 2.x

## Features
```
- expressjs
- mithril 2.x
- parceljs
- sequelizejs
- mysql
```

## How to start
1. Run `npm install`
2. Rename the database name in `sequelize/config/config.json` to your database
3. Run `npm start`, server will run on `localhost:8001`
